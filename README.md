
# Hospitering 2025

Eit prosjekt for å lære git og til bruk for ospitering.



## Run Locally

For å kjøre dette prosjektet lokalt må du først clone repoen
```bash
  git clone git@github.com:BriFBoy/Hospitering-2025.git
```
Deretter åpner do repoen i vscode eller lignende og opner html dokumentet i ein netleser
